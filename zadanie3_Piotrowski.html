Temat: Cechy modyfikowania formatowania w HTML oraz CSS

Nowe możliwości CSS w porównaniu z HTML:

Selektory: CSS oferuje szeroki zakres selektorów, co pozwala na precyzyjne określenie, które elementy HTML mają być stylizowane.

Kaskadowość: CSS pozwala na tworzenie kaskadowych reguł stylów, gdzie można nadpisywać style dla konkretnych elementów, a także dziedziczyć style z wyższego poziomu.

Właściwości animacji i przejść: CSS umożliwia tworzenie animacji i przejść bez potrzeby używania dodatkowych skryptów, co pozwala na płynne efekty wizualne.

Responsywność: Media queries w CSS umożliwiają dostosowanie stylów do różnych urządzeń i rozmiarów ekranów, co pozwala na tworzenie responsywnych stron internetowych.
